# מדריך התקנה - TaskWhisper

כדי להתחיל להשתמש ב-TaskWhisper, עקבי אחר הצעדים הפשוטים הבאים:

## דרישות מערכת
* דפדפן מעודכן (Chrome, Firefox או Safari).
* גישה למיקרופון.
* חשבון פעיל ב-Google או GitHub.

## שלבי ההתקנה

### 1. התקנת תוסף הדפדפן
הורידי את התוסף מחנות האפליקציות של הדפדפן שלך וחברי אותו לסרגל הכלים.

### 2. הגדרת פקודת ההפעלה
לאחר ההתקנה, לחצי על האייקון של האוזן הקטנה והגדירי את "מילת הקסם" שלך (למשל: "היי וויספר").

### 3. חיבור יומנים
עברי להגדרות ובחרי ב-"Integrations" כדי לחבר את לוח השנה שלך.
